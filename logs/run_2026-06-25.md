### Technical note on Playwright:
Python playwright was installed successfully and can launch Chromium. However, the proxy passes
HTTPS through a tunnel that Playwright/Chromium cannot negotiate — all external HTTPS requests
fail with ERR_TUNNEL_CONNECTION_FAILED. The `requests` library handles this correctly through the
pre-configured proxy CA bundle.
